# 📋 java-kanban-jeen

Трекер задач, работающий по принципу канбан-доски. В нём можно создавать задачи, разбивать их на подзадачи и объединять в эпики. Статус эпика рассчитывается автоматически на основе состояния подзадач. Все данные сохраняются в CSV-файл и восстанавливаются при перезапуске. У проекта есть собственный HTTP-интерфейс для работы через браузер или сторонние приложения.

---

## ✨ Функциональность

- Создание, обновление, удаление и получение задач
- Управление эпиками и подзадачами
- Автоматический расчёт статуса эпика
- Сохранение и восстановление состояния из CSV-файла
- HTTP-интерфейс для взаимодействия с менеджером задач
- Покрытие кода юнит-тестами
